## Install Signalize

Use NPM, PNPM or Yarn:
```bash
npm i signalizejs

yarn add signalizejs

pnpm i signalizejs
```

## Initialization

Create `src/assets/layout.js` and add the following code into it:
```js
import Signalize from 'signalizejs';

export const signalize = new Signalize();

// Define global variable
window.signalize = signalize;
```

Open your layout file (like `src/layouts/main.astro`) and add the following code into it:
```html
---
// 1. Get signalizejs url
import signalize from 'signalizejs?url';

// 2. Get event module url. This applies for all modules you will need.
import eventModuleUrl from 'signalizejs/event?url';

// 3. Define importmap
const importMap = JSON.stringify({
	imports: {
		// 4. Configure necessary modules
		'signalizejs/event': eventModuleUrl,
		// Other modules you need
	}
});
---

<html>
	<head>
		<!-- 5. Embed the import map -->
		<script type="importmap" :html={importMap}></script>
		<!-- 6. Create a global Signalize instance -->
		<script>
			import "../assets/layout.js";
		</script>
	</head>
	<body>
		<slot />
	</body>
</html>
```

## Use Signalize from other places
Astro.build uses Vite and bundles javascript under the hood.

This is not what we want, becaus we want to use ES modules and import maps.

Therefore, we are not going to import Signalize from layout but from the global window object.
```html
---
// src/pages/index.astro
import MainLayout from '../layouts/main.astro';
---
<MainLayout>
	<script>
		import type { signalize } from '../assets/layout.js';
		import type { on } from 'signalizejs/event';

		// 7. Signalize instance will be used from window object
		// Types are used from source files
		const { on } = await signalize.resolve<{ on: on }>('event');
	</script>
</MainLayout>
```
