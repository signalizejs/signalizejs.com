## Installation
```js
import Signalize from 'signalizejs';
import heightPlugin from 'signalizejs/height';

const { height } = new Signalize({ heightPlugin() });
```

## API

### height
Get height of an element or document
```js
const { height } = signalize;

height(document.querySelector('#element'));
```
