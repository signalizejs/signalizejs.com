## Installation
Add a public path for `signalizejs/is-visible` into your import map.

Then you can resolve it.

```js
const { isVisible } = await signalize.resolve('is-visible');
```

## API

### isVisible
Check if element is visible. Method returns `true` or `false`.
```js
isVisible(document.querySelector('#element'));
```
