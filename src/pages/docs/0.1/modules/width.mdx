## Installation
The `width` plugin comes with the Signalize default installation. You just need to import it.

```js
import Signalize from 'signalizejs';
import widthPlugin from 'signalizejs/width';

const { width } = new Signalize({
	plugins: [ widthPlugin() ]
});
```

## API

### width
Return a width of an element or document

```js
const { width } = signalize;

width(document.querySelector('#element'));
```
