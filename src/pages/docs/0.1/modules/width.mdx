## Installation

Add a public path for `signalizejs/width` into your import map.

Then you can resolve it.

```js
const { width } = await signalize.resolve('width');
```

## API

### width
Return a width of an element or document

```js
const { width } = signalize;

width(document.querySelector('#element'));
```
