## Installation
<ImportMapDependencies module="dom/ready" />

```js
await signalize.resolve('dom/ready');
```

This module doesn't expose any functionality. It only triggers `dom:ready` event. See below.

## Usage
This plugin ads a `dom:ready` event, you can listen to through the [on](/docs/modules/event#on) function.

```js
const { on } = await signalize.resolve('event');

on('dom:ready', () => {
	// Do something
});
```

## Dispatched events
You can listen to these events through the [on](/docs/modules/event#on) function.
`dom:ready`: Triggered when the DOM is [ready](https://developer.mozilla.org/en-US/docs/Web/API/Document/DOMContentLoaded_event).
