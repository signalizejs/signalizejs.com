## Installation
<ImportMapDependencies module="logger" />

```js
import loggerModule from 'signalizejs/logger';

const signalize = new Signalize({
	modules: [
		[
			'logger',
			loggerModule,
			{
				// Url where the logs will be send
				url: 'https://example.com/api/log-client-error',
				// Optional
				// What type of info should be send
				// Default is "error" only
				levels: ['log', 'warning', 'error']
			}
		]
	]
})
```

## Usage
The logger plugin starts automatically and doesn't have any API:
- After the module setup, logger will automatically start sending requests to the configured endpoint.
- The default level that is send is `error` only.
- You can change it by passing `levels` during the initialization.
